# Netflix Movie Data Analysis using Python

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on a Netflix movie dataset using Python. The analysis focuses on understanding movie trends, genre distribution, popularity, ratings, vote counts, and release patterns. Various visualizations are created to uncover meaningful insights from the dataset.

---

## Objectives
- Clean and preprocess movie data.
- Analyze genre distribution.
- Explore popularity and rating trends.
- Study vote counts and audience engagement.
- Examine movie release trends over time.
- Generate actionable insights using data visualization.

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Dataset Features

| Column Name | Description |
|------------|-------------|
| Release_Date | Movie release date |
| Title | Movie title |
| Overview | Movie description |
| Popularity | Popularity score |
| Vote_Count | Number of votes received |
| Vote_Average | Average rating |
| Original_Language | Original movie language |
| Genre | Movie genre |
| Poster_Url | Movie poster URL |

---

## Data Cleaning Steps
- Checked for missing values.
- Removed unnecessary columns.
- Converted release date to datetime format.
- Processed genre information.
- Handled duplicate records.
- Performed feature engineering where required.

## Analysis Performed

### 1. Genre Distribution Analysis
- Identified the most common movie genres.
- Compared genre frequencies.

### 2. Popularity Analysis
- Examined highly popular movies.
- Studied popularity trends across genres.

### 3. Rating Analysis
- Analyzed vote averages.
- Categorized movies based on ratings.

### 4. Vote Count Analysis
- Investigated audience engagement through vote counts.
- Compared votes across rating categories.

### 5. Release Trend Analysis
- Explored movie releases over different years.
- Identified growth in movie production.

---

## Key Insights

### Genre Insights
- Drama is the most dominant genre in the dataset.
- Comedy, Action, and Thriller are highly represented.
- Western, Documentary, and TV Movie genres have the lowest representation.

### Vote Analysis Insights
- Average-rated movies received the highest number of votes.
- Popular movies also attract strong audience engagement.
- Voting activity is fairly balanced across rating categories.

### Release Trend Insights
- Movie production has increased significantly over the years.
- Most movies in the dataset were released after 2000.
- Recent years contribute the largest share of movie releases.
